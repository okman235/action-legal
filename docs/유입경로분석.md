# ActIon 유입경로(획득 채널) 분석

> 작성일: 2026-07-17
> 대상 서비스: **ActIon** — AI 기반 연기·발음·감정·매체연기 연습 앱 (iOS)
> 목적: 신규 이용자를 확보할 수 있는 유입 채널을 조사·정리하고, 채널별 우선순위와 실행 방안을 제시한다.

---

## 1. 서비스·타깃 요약

ActIon은 회원가입 없이 사용하는 iOS 앱으로, 다음 기능을 제공한다.

- AI 기반 연기 분석 및 피드백 (촬영 영상·프레임 분석)
- 발음(딕션) 연습 및 음성 인식 기반 교정
- 감정·매체연기 분석
- AI 상대역·감독 음성(TTS)으로 대사 연습
- 무료 체험 + 구독(무제한) 모델

### 핵심 타깃 (Persona)

| 세그먼트 | 특징 | 니즈 |
|---|---|---|
| 연기 입시생 | 예대·연영과 입시 준비, 10대 후반~20대 초반 | 반복 연습, 객관적 피드백, 비용 부담 |
| 배우 지망생·신인 배우 | 오디션 상시 준비, 20대 | 오디션 대사 연습, 상대역 부재 해결 |
| 연기 전공생 | 대학 연극영화과 재학생 | 과제·실습 보조, 발성·딕션 훈련 |
| 취미 연기·스피치 | 발표·면접·스피치 향상 희망 | 발음 교정, 자신감 |

> 공통 페인포인트: **상대역이 없어 혼자 연습하기 어렵다**, **연기·발음에 대한 객관적 피드백을 받기 어렵다**, **레슨 비용이 비싸다**. → ActIon의 메시지는 이 세 가지를 해소하는 방향으로 정렬한다.

---

## 2. 유입 채널 후보 전체 지도

크게 4개 그룹으로 분류한다.

1. **오가닉 검색 / 스토어** — ASO, App Store 검색, 웹 검색
2. **소셜·콘텐츠** — 숏폼(릴스·틱톡·쇼츠), 유튜브, 인스타그램
3. **커뮤니티·제휴** — 연기입시 카페, 학원·대학 제휴, 오디션 커뮤니티
4. **페이드(유료광고)** — Apple Search Ads, Meta Ads, 인플루언서

---

## 3. 채널별 상세 분석

### 3.1 App Store 검색 + ASO (앱 스토어 최적화)

- **성격**: 오가닉, 구매 의도 가장 높음(검색해서 들어오는 사람은 이미 니즈 보유).
- **핵심 키워드 후보**: `연기 연습`, `연기 앱`, `대사 연습`, `발음 연습`, `딕션`, `연기 입시`, `오디션 연습`, `상대역`, `연기 피드백`.
- **실행**:
  - 앱 이름/부제에 핵심 키워드 1~2개 자연스럽게 포함.
  - 스크린샷 첫 장에 "혼자서도 상대역과 대사 연습" 같은 핵심 가치 문구.
  - 첫 화면 3초 안에 "AI가 연기를 분석해준다"는 차별점 노출.
- **측정**: App Store Connect > 앱 분석의 "검색" 유입 비중, 키워드별 노출·전환.
- **우선순위**: ★★★★★ (비용 대비 효율 최상, 즉시 착수)

### 3.2 숏폼 콘텐츠 (Instagram 릴스 / TikTok / YouTube Shorts)

- **성격**: 오가닉/바이럴, 타깃(10~20대 연기 지망생)이 가장 많이 머무는 곳.
- **콘텐츠 아이디어**:
  - "AI한테 내 연기 점수 매겨봤다" — 실제 분석 결과 리액션.
  - "상대역 없어서 AI랑 대사 맞춰봄" — 앱 사용 장면.
  - "연기 입시생이 매일 하는 딕션 루틴" — 발음 연습 활용.
  - Before/After 발음·감정 피드백 비교.
- **핵심**: 앱 화면 자체가 콘텐츠가 되게 만든다(스크린 레코딩 + 리액션).
- **측정**: 프로필 링크(App Store 링크)에 캠페인 파라미터, 게시물별 유입.
- **우선순위**: ★★★★★ (타깃 밀집도·바이럴 잠재력 최고, 초기 집중 투자)

### 3.3 YouTube (롱폼 + 연기 유튜버 협업)

- **성격**: 신뢰·정보 채널. 연기 팁·오디션 정보 검색 수요 존재.
- **아이디어**:
  - 연기 레슨/오디션 팁 유튜버와 협업하여 "AI로 연습 보완하기" 소개.
  - 자체 채널로 "연기 연습 루틴", "발음 교정법" 등 하우투 콘텐츠 → 앱 자연 노출.
- **우선순위**: ★★★☆☆ (효과는 크나 제작·협업 리드타임 김)

### 3.4 네이버 카페·커뮤니티 (연기입시·배우 지망 커뮤니티)

- **성격**: 국내 연기 지망생이 실제로 정보를 얻는 핵심 오가닉 공간.
- **대표 공간**: 연기입시 관련 대형 카페, 대학별 연영과 준비 모임, 오디션 정보 커뮤니티.
- **실행**:
  - 광고성 도배가 아니라 "혼자 연습 어떻게 하냐" 류 실제 고민글에 도움되는 후기·팁으로 접근.
  - 무료 체험 기반 후기 이벤트.
- **주의**: 커뮤니티는 노골적 홍보에 민감 → 진성 후기·유용성 우선.
- **우선순위**: ★★★★☆ (전환 의도 높은 진성 타깃, 신뢰 기반 접근 필요)

### 3.5 학원·대학·기관 제휴 (B2B2C)

- **성격**: 오프라인 접점을 통한 대량 유입.
- **대상**: 연기학원, 입시학원, 대학 연극영화과, 오디션 에이전시.
- **실행**:
  - 학원 수강생 대상 프로모션 코드/기간 무료 제공(연습 보조 도구로 포지셔닝).
  - 강사가 과제 피드백 보조로 쓰도록 제안.
- **우선순위**: ★★★☆☆ (전환율 높지만 영업 리드타임 길고 초기엔 소규모 파일럿 권장)

### 3.6 인플루언서 마케팅 (배우·연기 크리에이터)

- **성격**: 페이드/협찬. 3.2·3.3의 협업 버전.
- **실행**: 마이크로 인플루언서(연기 지망 팔로워 보유) 다수와 소액 협업 → 릴스/쇼츠 UGC 확보.
- **측정**: 인플루언서별 고유 링크/코드.
- **우선순위**: ★★★★☆ (숏폼 전략과 시너지, 예산 확보 시)

### 3.7 Apple Search Ads (ASA)

- **성격**: 페이드 검색. 스토어 검색 상단 노출.
- **실행**: 3.1에서 도출한 고전환 키워드에 소액 입찰로 시작 → CPA(설치당 비용) 검증 후 확장.
- **측정**: ASA 대시보드 + 키워드별 전환.
- **우선순위**: ★★★★☆ (오가닉 검색 검증 후 즉시 확장 채널)

### 3.8 Meta Ads (Instagram/Facebook)

- **성격**: 페이드 소셜. 관심사·유사타깃 기반 확장.
- **실행**: 숏폼에서 성과 좋았던 크리에이티브를 광고 소재로 전환(오가닉 검증 → 페이드 확장).
- **우선순위**: ★★★☆☆ (오가닉 콘텐츠 성과 데이터 확보 후 진행)

## 5. 단계별 실행 로드맵

### 1단계 — 기반 다지기 (착수 즉시, 비용 최소)
- ASO 정비(이름/부제/스크린샷/키워드).
- App Store Connect 앱 분석에서 **유입 소스별 데이터 수집 체계 확립**.
- 마케팅 링크에 캠페인 파라미터 규칙 정립(어느 채널에서 왔는지 구분).

### 2단계 — 오가닉 콘텐츠 엔진 (1~2단계 병행)
- 숏폼 주 2~3회 발행, 앱 화면 기반 콘텐츠 실험.
- 네이버 카페 진성 후기/팁 활동.
- 성과 좋은 콘텐츠 유형·훅(hook) 데이터 축적.

### 3단계 — 페이드 확장 (오가닉 검증 후)
- Apple Search Ads: 검증된 고전환 키워드에 소액 시작.
- 성과 좋은 숏폼 소재를 Meta/인플루언서로 확장.

### 4단계 — 제휴 (병렬 파일럿)
- 학원·연영과 1~2곳 파일럿 → 전환·리텐션 검증 후 확대.

## 6. 유입경로 측정 방법 (데이터 수집 체계)

채널 분석이 "감"이 아니라 데이터가 되려면 아래를 갖춰야 한다.

1. **App Store Connect 앱 분석**
   - 소스 유형(App Store 검색 / 브라우징 / 웹 참조 / 캠페인)별 페이지 조회·다운로드 확인.
   - "캠페인" 데이터를 쓰려면 마케팅 링크에 **캠페인 토큰(pt·ct·mt)** 을 붙인 App Store 캠페인 링크 사용.
2. **채널별 고유 링크**
   - 인플루언서·게시물·커뮤니티마다 서로 다른 캠페인 링크/프로모 코드 부여 → 어디서 왔는지 구분.
3. **인앱 첫 화면 설문(선택)**
   - "어떻게 알고 오셨나요?" 1문항으로 오프라인·구전 유입 보완(스토어 데이터의 사각지대 보완).
4. **핵심 지표(KPI)**
   - 채널별: 노출 → 페이지 조회 → 다운로드 → 무료체험 시작 → 유료 전환(퍼널).
   - 채널별 **CAC(획득 비용)** 와 **전환율**로 우선순위를 지속 재조정.

> 참고: 이 저장소의 `index.html`(개인정보처리방침)은 앱 심사·신뢰용 정적 페이지다. 별도의 마케팅 랜딩 페이지를 운영한다면 그 페이지에 웹 분석(리퍼러/UTM 수집)을 붙여 웹→스토어 유입까지 추적하는 것을 권장한다. (원하시면 랜딩 페이지 + 분석 세팅도 별도로 구성해 드릴 수 있습니다.)

## 7. 요약 (Executive Summary)

- **가장 먼저 할 일**: ASO 정비 + 유입 데이터 수집 체계 확립(비용 0, 효과 즉시).
- **초기 집중 채널**: App Store 검색(ASO)과 숏폼 콘텐츠 — 타깃 밀집도와 비용 효율이 가장 높다.
- **검증 후 확장**: 오가닉에서 검증된 메시지·키워드를 Apple Search Ads·인플루언서로 확장.
- **차별화 메시지**: "상대역 없이 혼자, AI가 연기를 분석하고 상대역이 되어준다" — 모든 채널 크리에이티브의 공통 훅.
- **핵심 원칙**: 모든 채널에 **측정 링크**를 붙여 CAC·전환율 데이터로 예산을 재배분한다.
